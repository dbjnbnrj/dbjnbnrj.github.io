StochSS IDE September 2014 - Present
Graduate Student Researcher Santa Barbara, CA
StochSS is an IDE for discrete stochastic simulations. I created 3D visualizations of these models using WebGL, three.js and Python. While creating these models I improved the overall rendering time of models using caching, animated the models over a time series and used volume rendering techniques to give users a better indication of the density and concentration of species in the model.

    • Predicting the Impact of New Businesses on Existing Ones Grad Project
    Proximity of businesses in cities encourages competition. This increases when a new business opens in the same area. Using data provided by the Yelp 2015 dataset challenge, we predicted the impact new businesses have on the ratings of existing ones in the same locality.

    Practical Approach to Session Types Grad Project
    I built a linear type checker based on Vasconcelos' Fundamental Session Types Algorithmic Type Checker. Session Types can be used as type systems for secure network protocols.
    Learning Chess Heuristics Using Neural Networks Grad Project
    In this project I used a three layered Convolutional Neural Network to predict moves in chess based on recogniing patterns from previous players games. My dataset consisted of 500 games from the FICS Games Database. Using this network I was able to predict the next best chess move with 33% accuracy.
    Ramsey Counterexample Generation Grad Project
    Ramsey Graphs are complete graph with specific combinatorial properties. In this project we used a number of cloud computing platforms (AWS, Azure etc.) and Super Computers (Condor, Xsede) in order to generate a counterexample for Ramsey number 7.
    Group:@dbjnbnrj, @varks, @rathodsachin20 and @sahaj
    Paxos Based Banking Application Grad Project
    The Paxos Algorithm is a protocol for solving consensus in a network of unrelaible computers. This project implements a simple banking system that handles operations like "deposit", "transfer", or "get-balance" in a system of computers. The protocol works in a way that ensures consensus across all machines regardless of machine failure or invalid operations.
    Group: @dbjnbnrj, @micang
